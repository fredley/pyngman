Pyngman

Optimal Hangman-Playing Python Script

Download this project as a .zip file Download this project as a tar.gz file

Play Hangman Perfectly

This script processes a dictionary and will allow you to make optimal guesses in the game hangman. You can also work out what words are the 'hardest' to guess, which will give you the best chance of hanging your opponent.

How to use pyngman

  1. Grab a dictionary - you'll need a dictionary to play. Tested with Pyxidium's British English dictionary. A full list of dictionaries is available on the Open Office Wiki, note that non-English characters have not been tested.

  2. Initialise pyngman - pyngman needs to process the dictionary to generate word lists, so you need to initialise it before you can do anything. From the command line, run:
    $ python pyngman.py -init en_GB.zip
    If you want to include proper nouns (any words containing capital letters), use -p:
    $ python pyngman.py -init -p en_GB.zip

  3. Calculate Optimal guesses - we're ready to start playing hangman! Pass pyngman a state, and a list of letters you've tried and it'll tell you what your optimal next guess is:

$ python pyngman.py -state ..e.. est
> Your best next guess is: a

Generating statistics

hangman.py emulates hangman, and together with test.py and a bit of fiddling, you can easily walk through all possible games of hangman to generate statistics. Be warned though, full walks of the dictionary take hours to complete!
